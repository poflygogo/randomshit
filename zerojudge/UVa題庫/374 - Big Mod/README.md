# 題意解析
有一數學算式 $R=B^P\bmod M$

mod在數學中是求餘的意思，用python語法表達這公式的話就是 `R = B ** P % M`

題目會依序提供P、B、M的值，請輸出R的值作為答案，其中:
- 0 <= B <= 2147483647
- 0 <= P <= 2147483647
- 1 <= M <= 46340

> 對相當大的B、P、M請寫一個有效率的演算法來。

|範例輸入|範例輸出|
|----|----|
|3<br>18132<br>17<br><br>17<br>1765<br>3<br><br>2374859<br>3029382<br>36123|13<br>2<br>13195|
