# 關於前綴和
## 解釋
前綴和，顧名思義，就是一個數列中，從開頭到某個位置的所有數的和。形象地說，就像是一本書的目錄，每個目錄標題都告訴你從書首到該章節的所有頁數總和。

## 用途
- 加速區間查詢: 當我們需要頻繁地查詢一個數列中某個區間的元素總和時，使用前綴和可以大大提高查詢效率。
- 簡化計算: 很多問題都可以通過前綴和來轉換，使得問題的解決變得更加簡單。

## 舉例來說
有一數列[1, 3, 5, 7, 9]

它的前綴和就是[1, 4, 9, 16, 25]
